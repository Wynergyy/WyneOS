\# Getting Started with WyneOS Genesis



This guide explains how to install, run, and explore the WyneOS Genesis Framework.



\## 1. Prerequisites

\- Node.js (latest LTS recommended)

\- Git



\## 2. Install Dependencies

From the WyneOS root folder, run:



npm install



This installs TypeScript and local-only developer tooling.



\## 3. Run the Framework

To start WyneOS Genesis:



npm start



You should see logged output from the runtime, DataFabric, WyneID, GuardianMesh, and WYNE-UI modules.



\## 4. Run Tests

To execute the local test suite:



node Testing/run-tests.js



\## 5. Explore Modules

Each module is contained in its own folder and is fully transparent and inspectable.



This framework is safe, local, and under your direct control.
