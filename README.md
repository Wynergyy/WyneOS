\# WyneOS Genesis System

A safe, ethical, controlled foundational scaffold.



\## Principles

\- No networking

\- No autonomous behaviour

\- No analytics

\- No surveillance

\- No behavioural inference

\- Everything is user-controlled and fully transparent



\## Modules

WAR Runtime         - Basic execution and routing

DataFabric          - Local key-value storage

WyneID              - Identity object with consent model

GuardianMesh        - Safe manual event logging

WYNE-UI             - Simple UI elements



\## Running

npm install

npm start
